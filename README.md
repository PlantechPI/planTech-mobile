# PlanThech

## Visão Geral
O PlanThech é um aplicativo desenvolvido para capacitar agricultores no controle eficiente de suas culturas. Com o PlanThech, os agricultores podem monitorar suas plantações em tempo real, acessar um histórico detalhado de dados e gerenciar a irrigação de suas culturas de forma remota.
### Monitoramento em Tempo Real
O app oferece monitoramento contínuo, permitindo que os agricultores obtenham informações atualizadas sobre as condições de suas culturas. Sensores instalados nos campos coletam dados relevantes que são transmitidos diretamente para o aplicativo, proporcionando uma visão clara e precisa do estado das plantações.
### Histórico de Dados
Além do monitoramento em tempo real, o PlanThech armazena um histórico detalhado das informações coletadas. Isso permite aos agricultores analisar tendências, identificar padrões e tomar decisões informadas com base em dados históricos. Essa funcionalidade é crucial para planejar ações futuras e otimizar a produtividade agrícola.
### Controle de Irrigação
Uma das funcionalidades mais destacadas do PlanThech é a capacidade de controlar a irrigação das culturas de forma automatizada e remota. Os agricultores podem ajustar os horários e a quantidade de irrigação diretamente pelo aplicativo, garantindo que suas plantações recebam a quantidade adequada de água de acordo com as necessidades específicas do solo e das plantas.
### Benefícios
O PlanThech foi projetado para ser uma ferramenta intuitiva e acessível, mesmo para aqueles com pouca experiência em tecnologia. Seu design simples facilita a navegação e o uso, tornando a gestão agrícola mais eficiente e menos trabalhosa. Com o PlanThech, agricultores familiares podem melhorar significativamente a gestão de seus recursos, aumentar a produtividade e promover práticas agrícolas mais sustentáveis.

PlanThech é, portanto, uma solução completa e moderna para os desafios enfrentados pelos agricultores na gestão de suas culturas, oferecendo uma combinação poderosa de monitoramento, histórico de dados e controle de irrigação.

## Descrição
### Objetivos
Este app é parte de um projeto de faculdade com objetivo de automatizar a fertiirrgação de agricultores famíliares. 
### Desining
Todo o desining foi pensado para ser o mais simples e intuítivo possível para o agricultor.
### Base de dados
Este app é alimentado por informações de um banco de dados com informações coletadas por nossos sensores que captam informações reais. 


## Instalação
Para executar o aplicativo PlanThech localmente em seu ambiente de desenvolvimento, siga estas etapas:

1. Certifique-se de ter o Node.js e o npm instalados em seu sistema.
2. Clone este repositório em sua máquina local usando o Git:
$ git clone https://github.com/PlantechPI/planTech-mobile.git

3. Navegue até o diretório do projeto:
```sh
$ cd PlanThech
```

4. Instale as dependências do projeto:
```sh
npm install
```

6. Execute o projeto usando Expo:
```sh
$ npx expo start
```

7. Abra o aplicativo Expo Go em seu dispositivo móvel e escaneie o código QR fornecido no terminal para visualizar o aplicativo PlanThech em seu dispositivo.
